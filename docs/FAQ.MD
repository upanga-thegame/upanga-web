# Upanga FAQ

## Game

### What is Upanga: The Soul-Blade?
Upanga: The Soul-Blade is a tactical, turn-based fantasy RPG for Android where four heroes battle evil forces through the land of Azuma.


### What platform is Upanga being developed for?
Upanga is being developed for Android mobile devices.


### What stage of development is the game in?
Upanga is currently in pre-alpha development. Core systems such as movement, turn order, combat, UI, and save/load are still being built and tested.


### Will the game be free?
The game will be free to download. The first level, if you will, of the game will be playable. After completing the first objectives you may unlock the full Chapter 1 for a small price.


## Gameplay

### What kind of combat does Upanga use?
Upanga uses initiative, tactical turn-based combat on an isometric grid. Players command heroes, move across tiles, spend action points, and fight enemies. 


### Can I save the game?
The game is planned to support local, non-cloud saving using one save file. The goal is to allow players to save and load their current game state. Future state Google Cloud saves.


### Is Upanga online or offline?
Upanga is planned as a single-player game. Online features are not required at this stage.


## Release

### When will Upanga be released?
There is no final release date yet. The game is currently in development but targeting Fall of 2026.


### Will there be a playtest?
A playtest may be offered later when the game reaches a more stable alpha or beta stage.


### How can I get updates?
You can follow the website, social media, or join the newsletter once it is connected.
